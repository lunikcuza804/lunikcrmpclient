####                                               Description                                                     #### 
####################################################################################################################### 

You like the game GrandTheftAuto, its opportunities and freedom of action? But you would like something else? New? Mother? 
Then you need to plunge into the world of GTA Criminal Russia. 
Mod events taking place in today's days. 
In our project, you can find modern vehicles (domestic and foreign cars). 
* Interesting facts and places. 
* Various card from the plains to the lush fields russrih forests and mountains. Highly-detailed nature. 
* Realestichnaya atmosphere and weather effects, improved graphics visual effects. 
* New game features, such as the opportunity to ride on the Ferris wheel, an elevator into Antenkinskoy tower, 
go to clubs with a stylish music or get on the crazy and incendiary OpenAir'y. 
And much, much more.

Minimum system requirements: 

Windows 98/2000/XP/Vista/Windows 7 
Pentium 4 2.4GHz, 256 MB RAM 
NVidia GForce5 128 MB VRAM 
16 bit DirectX compatible sound card 
5.5 GB hard disk. DirectX 9.0 

Recommended system requirements: 

Windows XP / Vista / Windows 7 
Intel Core 2 Quad 2.4GHz, 3GB RAM 
NVidia 8800GT 512 MB video memory or higher 
16 bit DirectX compatible sound card 
6 GB for hard disk. DirectX 9.0 or higher 

CAUTION 
For normal operation of the game under OS Vista / Windows 7, you must perform the following steps: 
1) Start -> Control Panel -> User Accounts and Family Safety -> User Accounts 
2) Vybrayt click Change settings User Account Control " 
3) Delete the slider to the bottom and click "OK" 

Then: 
1) Start-> All Programs-> Accessories-> Right-click on the "Command Prompt" - Run of Adiministrator. 
2) prints the string 
"Bcdedit.exe / set nx AlwaysOff" or "bcdedit.exe / set (current) AlwaysOff" 
Press "Enter" 
3) Reboot your computer. 

Only in this case is guaranteed correct operation of our project. 

If you have problems with the game under OS Vista / Windows 7 
install file gta-sa.exe from the root of the game, compatible with Windows 98.
